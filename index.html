  • blocks-demo - an initial prototype of a 2D layout system for code

  • alternate example visualization Another iteration on a visualization for parse trees in ohm.
  • ohm-threed-visualization - an experimental 3D visualization of an ohm parse tree for a program
  • text falls apart - experiments with degenerating prose
